O período da tabela equivale a linha (varia entre 1 a 7 e os asteriscos); enquanto que os grupos da tabela (grupos 1A, 2A, 3B e afins) são as categorias dos elementos que os compõem (metais alcalinos, alcalinos terrosos, metais de transição,halogênios, etc). Para diferenciar período de grupo, o grupo se destaca com o tamanho de fonte maior.
